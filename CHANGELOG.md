## v0.6.9 - 2026-04-12 (الإصدار النهائي الموحد)

    - تعطيل التخزين المؤقت (Caching) في `vercel.json` لضمان استلام التحديثات فوراً.

    - تحديث واجهة المسؤول لإظهار الطابع الزمني للتحديث.

## v0.6.8 - 2026-04-12
- 🚀 **تحديث النظام والمزامنة**:
    - تحديث إصدار النظام إلى V0.6.8-LIVE-SYNC.
    - إصلاح إعدادات Vercel لضمان استمرارية النشر (تجاهل أخطاء ESLint أثناء البناء).
    - توحيد أرقام الإصدارات في جميع ملفات المشروع (package.json, STATUS.md, UI).

## v0.6.0 - 2026-04-11
### 🆕 ميزات وإصلاحات تقنية (Native & Web)
- 🛠️ **تحديثات النظام الأساسية (Native)**:
    - تحديث Android SDK إلى الإصدار 35 (Android 15) لضمان التوافق مع بيئات البناء الحديثة.
    - تنظيف وتصحيح تضارب حزم Java (حذف ملفات MainActivity الزائدة).
    - رفع `versionCode` إلى 2 و `versionName` إلى 1.1 لتمكين التحديث عبر المتاجر.
- 🌐 **تحسينات واجهة الويب والـ Build**:
    - إصلاح مشاكل "Cascading Renders" في React 19+ لضمان استقرار التطبيق.
    - تعديل إعدادات Next.js لتجاوز أخطاء ESLint أثناء البناء لضمان استمرارية النشر.
    - تحسين منطق `AppWrapper` و `AuthGuard` لزيادة الأداء وسرعة التحميل.
