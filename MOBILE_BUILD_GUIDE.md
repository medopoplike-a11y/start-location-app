# تعليمات بناء ملف APK الجديد

للحصول على أفضل أداء وصلاحيات كاملة، يرجى اتباع الخطوات التالية في Terminal الخاص بك:

1. **تثبيت الاعتماديات الجديدة**:
   ```bash
   npm install
   ```

2. **بناء المشروع للويب**:
   ```bash
   npm run build:mobile
   ```

3. **مزامنة الملفات مع الأندرويد**:
   ```bash
   npx cap sync android
   ```

4. **فتح المشروع في Android Studio**:
   ```bash
   npx cap open android
   ```

5. **من داخل Android Studio**:
   - انتظر انتهاء "Gradle Sync".
   - اذهب إلى القائمة `Build` > `Build Bundle(s) / APK(s)` > `Build APK(s)`.
   - بمجرد الانتهاء، سيظهر إشعار يحتوي على رابط `locate` للملف.

### ملاحظات هامة للنسخة الجديدة:
- تم تفعيل نظام **المزامنة الفائقة** (Real-time).
- تم تغيير الخلفية إلى **الفضي المطفي** (Matte Silver) العصري.
- تم إصلاح مشكلة **الخروج التلقائي** بعد تسجيل الدخول.
- تم إضافة طلب **الأذونات الكاملة** (الموقع، الكاميرا، التنبيهات) عند التشغيل.
